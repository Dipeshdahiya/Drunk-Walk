## 🍺 Drunk Walk

A fun and chaotic browser game where walking straight becomes the biggest challenge.

Drunk Walk is a casual physics-style balance game where the player controls a drunk character trying to move forward without falling. The unpredictable movement makes the gameplay funny, challenging, and addictive.

---

## 🎮 Gameplay

The objective is simple:

Walk as far as possible without losing balance.

Because the character is drunk, every step becomes unpredictable. Players must carefully control movement to avoid falling or drifting off the path.

The longer you survive, the harder the game becomes.

## 🕹 Controls

### Keyboard
- **Left Arrow / A** → Move left
- **Right Arrow / D** → Move right

## 🚀 How to Run the Game

1. Clone the repository

git clone https://github.com/yourusername/drunk-walk.git

2. Navigate to the project folder

cd drunk-walk

3. Open `index.html` in your browser

OR run a local server:

python -m http.server

Then open:

http://localhost:8000
